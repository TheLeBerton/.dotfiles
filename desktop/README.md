## Download fonts for hyprlock
```bash
mkdir -p ~/.local/share/fonts
wget -O ~/.local/share/fonts/SFProDisplay-Bold.otf "https://raw.githubusercontent.com/MrVivekRajan/Hyprlock-Styles/main/Style-2/Fonts/SF%20Pro%20Display/SF%20Pro%20Display%20Bold.otf"
wget -O ~/.local/share/fonts/steelfish-outline.otf "https://raw.githubusercontent.com/MrVivekRajan/Hyprlock-Styles/main/Style-2/Fonts/steelfish%20outline%20regular/steelfish.outline-regular.otf"
fc-cache -fv
```
